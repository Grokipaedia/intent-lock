# 🔒 intent-lock

Force AI to respect constraints.

## What it does

Blocks outputs that break your rules.

## Example

Input:
Should I buy AGIHQS.com?

Output:
❌ BLOCKED: violates constraints

## Why

AI follows prompts.

But it doesn’t respect boundaries.

intent-lock enforces them.
